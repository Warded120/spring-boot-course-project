Про нас

Опис програми

Система управління онлайн курсами іноземних мов:

  • Управління користувачами, курсами тощо
  • Обробка запитів
  • Симуляція реального додатку

Стек технологій

Цей додаток побудований з використанням таких технологій:

  • Back-End: Java, Spring Boot, PostgreSQL
  • Front-End: HTML, CSS, Thymeleaf
  • Інструменти: Maven, Git, Docker

Для користувача

Даний пункт містить додаткову інформацію для тих, хто тестує додаток:

  • Паролі всіх користувачів зашифровані, але, за вмовчуванням, пароль для будь-якого користувача "Test123"
